# 训练实验计划表

## 无人机模型

| 实验编号 | 内容 | 输入数据 | 目标产物 | 通过标准 |
| --- | --- | --- | --- | --- |
| UAV-E0 | 小样本 pipeline 测试 | 少量切片与标签 | 可启动训练、验证和预测 | 流程可跑通，不看指标高低 |
| UAV-E1 | RGB 航拍 baseline | RGB 航拍图或切片 | baseline 权重与日志 | 输出验证结果和预测样例 |
| UAV-E2 | 切片训练 baseline | 高分辨率切片 | 小目标检测对比 | 分析漏检样本 |
| UAV-E3 | 伪 RGB 多光谱/指数图训练 | NDVI/NDRE/GNDVI 等 3 通道图 | 多光谱输入对比 | 与 RGB baseline 对比 |
| UAV-E4 | 大输入尺寸训练 | 1024 或 1280 输入 | 小目标改进实验 | 记录速度和显存消耗 |
| UAV-E5 | 小目标补样后复训 | 补充后的 v0.2 数据 | 复训权重 | 观察漏检是否降低 |
| UAV-E6 | 外部地块测试 | 未参与训练的新地块 | 泛化报告 | 输出误检漏检分析 |

## 手机模型

| 实验编号 | 内容 | 输入数据 | 目标产物 | 通过标准 |
| --- | --- | --- | --- | --- |
| PHONE-E0 | 小样本 pipeline 测试 | 少量近景图片与标签 | 可启动训练、验证和预测 | 流程可跑通 |
| PHONE-E1 | 近距离 RGB baseline | 手机 RGB 图片 | baseline 权重与日志 | 输出验证结果和预测样例 |
| PHONE-E2 | 数据增强对比 | baseline 数据 | 增强实验对比 | 分析过强增强是否伤害颜色特征 |
| PHONE-E3 | 类别均衡后复训 | 补齐少数类后的数据 | 复训权重 | 观察少数类 AP 和召回 |
| PHONE-E4 | 低质量图片鲁棒性测试 | 模糊、弱光、遮挡图片 | 鲁棒性报告 | 输出失败原因 |
| PHONE-E5 | 外部手机/新地块测试 | 未见过设备或地块 | 泛化报告 | 输出误检漏检分析 |
| PHONE-E6 | 上传接口联调测试 | 后端上传图片 | 接口联调记录 | 返回统一 JSON |
