# Project Goal

你是一名资深全栈工程师。

你的任务不是生成 Demo 页面，而是开发一个完整的工程化平台。

项目名称：

> AI 水稻病虫害智能分析平台（Competition Demo Edition）

这是一个用于比赛演示的视频演示平台。

## 项目定位

本项目不是论文 Demo，不是算法 Demo，也不是只有上传图片和返回结果的识别网页。

本项目必须体现为一个完整的软件平台：

- 首页 Dashboard
- 检测中心
- 历史管理
- AI 助手
- 系统设置
- Mock Backend
- Make 自动化
- 本地知识库
- 可替换的检测服务
- 可扩展的数据服务

病虫害识别只是平台中的一个核心模块，不是项目全部。

## 最终目标

项目最终需要达到：

> 可以稳定录制比赛视频。

不是：

> 模型真实可部署。

因此允许：

- Mock Detection
- Template AI
- Preset JSON
- Fake History
- Local Knowledge Base
- Mock Make Workflow
- Demo Mode

禁止为了真实性牺牲开发效率、演示稳定性和工程完整度。

## 平台必须体现

- 现代农业
- AI 分析
- 工程化
- 平台化
- 可扩展
- 高完成度
- 比赛演示友好

## 优先级

优先保证：

1. 页面完整
2. 交互优秀
3. 视觉高级
4. 运行稳定
5. 工程结构规范
6. 后续可替换真实 YOLO、数据库、LLM、Make

Codex 每次开发前必须牢记：

> 这是一个平台，不是一个识别网页。
