# Frontend Emergency Scope

本次只做比赛/演示前端急救，不做大规模重构。

## 修改目标

- 将首页从功能展示改为“今日巡检三步开始”。
- 将主演示路径固定为：工作台 -> 宿迁协同巡检 -> UAV 任务 -> 指数分析 -> 手机复查 -> 巡检报告。
- 收缩左侧导航，把非主线功能弱化到高级功能区。
- 给 Dashboard 增加部分接口失败兜底，避免单个接口失败导致整页不可用。
- 给图像检测页增加“使用示范图片检测”，避免现场必须临时找图。
- 明确标识演示分析、兜底数据和非生产诊断边界。

## 不在本次范围

- 不更换技术栈或 UI 框架。
- 不引入 React Router。
- 不删除已有业务页面。
- 不修改后端接口路径。
- 不把桌面工作台硬改成完整移动端响应式。
- 不提交本机网络修复脚本，例如仓库根目录的 `fix-github-hosts.ps1`。

## 运行方式

```bash
npm install
npm run dev
```

构建验证：

```bash
npm run build
```
